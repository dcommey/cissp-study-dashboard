# CISSP Practice Quiz - Video 1: Introduction

Answer the following questions based on the introductory material. Think like a manager!

---

### Question 1
An administrator is reviewing a security policy and notices that the password policy is outdated. The administrator's primary role is technical. If they want to align with the CISSP philosophy, which of the following is their best initial step?

A. Modify the password complexity settings on the Active Directory Domain Controller.  
B. Draft a revised policy and present it to the IT Security Manager for approval.  
C. Inform the senior leadership team directly of the security gap.  
D. Consult the security baseline documentation and wait for the annual review cycle.

### Question 2

A. They must be able to configure firewalls, write access lists, and perform penetration tests.

### Question 3

### Question 4
