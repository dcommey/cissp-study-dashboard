# Quiz: Security Operations

## Question 1
A financial institution implements emergency administrative accounts that require two separate managers to collaborate to access the recovery console. The password for the recovery account is divided into two halves, with Manager A holding the first half and Manager B holding the second. Neither manager knows the other's half. Which security principle does this implementation represent?

- A) Need to Know
- B) Split Knowledge
- C) Job Rotation
- D) Least Privilege

## Question 2
A multinational corporation has thousands of employees who access corporate cloud resources in AWS and GCP using company-issued tablets and smartphones. The CISO wants to ensure that in the event a device is lost or stolen, the security operations center (SOC) can immediately delete all corporate data from the device over the air. Which technology must the CISO deploy to enable this capability?

- A) Configuration Management Database (CMDB)
- B) Mobile Device Management (MDM)
- C) Privileged Access Management (PAM)
- D) Security Information and Event Management (SIEM)

## Question 3
A system administrator proposes a critical security change to close an open port on the database server to reduce its attack surface. The Change Advisory Board (CAB) has formally reviewed and approved the change request. According to formal change management procedures, what step must the administrator perform immediately BEFORE implementing and scheduling the change in the production environment?

- A) Document the final state in the configuration management database (CMDB)
- B) Obtain sign-off on the residual risk from senior executives
- C) Test the change in an isolated, non-production environment
- D) Establish a new baseline image for the database server

## Question 4
A data center manager is calculating hardware reliability metrics to plan for the lifecycle replacement of server infrastructure. The manager needs to distinguish between components that can be repaired (like a modular power supply or server chassis) and components that cannot be repaired and must be discarded upon failure (like solid-state drives or backup tape media). Which metrics should the manager apply?

- A) MTBF for repairable components; MTTF for non-repairable components
- B) MTTF for repairable components; MTBF for non-repairable components
- C) MTTR for repairable components; RTO for non-repairable components
- D) MTD for repairable components; ALE for non-repairable components

## Question 5
A vulnerability scanner identifies a critical vulnerability in a legacy software application running on an internal network. The application vendor has gone out of business, so no security patch is available. The organization's IT director implements firewall rules to restrict access to the application but cannot patch the code. The CISO decides to keep the application online. What is the remaining security exposure called, and who is responsible for it?

- A) Inherent risk, which is the responsibility of the system administrator
- B) Total risk, which is the responsibility of the security operations team
- C) Transferred risk, which is the responsibility of the firewall vendor
- D) Residual risk, which is the responsibility of senior management
