# Quiz: Preventing Attacks

## Question 1
A security operations team is deploying a system of decoy virtual machines that simulate the organization's demilitarized zone (DMZ). The systems are intentionally configured with known vulnerabilities and populated with fake customer records. The primary goal is to distract attackers from real networks and capture their attack tools. What is this security mechanism called, and what legal constraint must the team review?

- A) Honeypot; they must ensure they do not perform man-in-the-middle decryption.
- B) Honeynet; they must ensure they do not actively entice or encourage illegal actions (avoiding entrapment).
- C) Sandbox; they must check if the file hashes are uploaded to the cloud database.
- D) Unified Threat Management (UTM); they must comply with the Acceptable Use Policy (AUP).

## Question 2
An IT director wants to secure corporate laptops against unauthorized software installations and shadow IT. The director wants to implement a control where only corporate-approved software packages can run on the endpoints, and all other applications are blocked by default. Which control provides the highest level of security for this requirement?

- A) An application deny list (blacklist)
- B) An application allow list (whitelist)
- C) Signature-based anti-malware scanning
- D) Device posture validation

## Question 3
A security auditor recommends that the organization secure its audit logs so that a hacker who compromises a domain controller cannot modify or delete the local log files to hide their activities. Which technology should the security team implement to guarantee the integrity and non-repudiation of these logs?

- A) Flow-based NetFlow sampling
- B) WORM (Write Once, Read Many) storage
- C) SOAR playbooks and runbooks
- D) Centralized Syslog-ng over UDP

## Question 4
A Security Operations Center (SOC) is deploying a SOAR platform to automate its response to malware alerts. The team writes a detailed procedural document outlining how to verify the alert, isolate the host, and notify owners. They then translate this document into a series of automated scripts that the SOAR platform executes dynamically. What are the document and the automated script called, respectively?

- A) Runbook and Playbook
- B) Playbook and Runbook
- C) Baseline and Standard
- D) Threat Feed and Access Control List (ACL)

## Question 5
A security analyst wants to map the techniques used by an advanced persistent threat (APT) group against the organization's current defenses. The analyst needs to reference a comprehensive, non-sequential database of observed threat actor behaviors, tactics, and procedures (TTPs). Which framework should the analyst consult?

- A) The Cyber Kill Chain
- B) The MITRE ATT&CK Matrix
- C) ISO 27002
- D) Unified Threat Management (UTM)
