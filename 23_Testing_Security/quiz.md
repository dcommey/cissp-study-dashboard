# Quiz: Testing Security

## Question 1
A cloud-based SaaS provider is undergoing a third-party audit to assure its enterprise clients that its security and privacy controls are designed properly and operate effectively over a six-month period. The SaaS provider wants to protect the proprietary configuration details within the report, meaning the final report must be confidential and shared only under a Non-Disclosure Agreement (NDA). Which type of audit report should the SaaS provider request?

- A) SOC 1 Type 2
- B) SOC 2 Type 1
- C) SOC 2 Type 2
- D) SOC 3 Type 2

## Question 2
A security analyst needs to scan an external target server for open ports. To evade detection and avoid revealing the scanning machine's IP address to the target's intrusion detection systems (IDS), the analyst decides to perform a scan that exploits the predictable IP Identification (IPID) field of an inactive third-party system. Which scan type should the analyst configure?

- A) TCP Connect Scan
- B) TCP SYN Scan
- C) TCP ACK Scan
- D) Idle (Zombie) Scan

## Question 3
A security engineer is performing a vulnerability assessment on a critical web application. The application's database is located on a private internal network segment and is not directly accessible from the internet. However, the engineer suspects that input validation is missing on the login page. Which tool should the engineer use to automatically discover and exploit potential SQL injection flaws?

- A) Nmap
- B) ZAP
- C) SQLmap
- D) Metasploit

## Question 4
An engineering manager is overseeing the development of software for a high-risk autonomous vehicle system. The security policy dictates that the code must undergo a rigorous, formal peer-review process where defect detection is recorded systematically through six distinct phases: Planning, Overview, Preparation, Inspection, Rework, and Follow-up. Which code review methodology meets this requirement?

- A) Fagan Inspection
- B) Software Walkthrough
- C) Static Application Security Testing (SAST)
- D) Generational Fuzzing

## Question 5
During a post-incident review, an incident response lead notices that firewalls, Active Directory servers, and database servers logged the same malicious event with timestamps that differed by up to five minutes. This time skew prevented the team from establishing a clear, chronologically accurate timeline of the attack. What is the most effective remediation?

- A) Implement a central Syslog server to normalize logs
- B) Configure NetFlow on core switches to track conversations
- C) Deploy a SIEM to perform dynamic parsing
- D) Implement Network Time Protocol (NTP) to synchronize system clocks
