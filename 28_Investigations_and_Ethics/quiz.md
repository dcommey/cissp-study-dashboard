# Quiz: Investigations and Ethics

## Question 1
A corporate legal team is preparing for a contract dispute lawsuit against a competitor. The team must identify, preserve, collect, and review all email correspondence and database records relevant to the case. Which industry-standard framework should the legal team follow to guide this e-discovery process?

- A) Fagan Inspection Model
- B) Electronic Discovery Reference Model (EDRM)
- C) Common Criteria (ISO 15408)
- D) MITRE ATT&CK Matrix

## Question 2
A digital forensics examiner is collecting evidence from a suspect's laptop. The examiner needs to extract a bit-stream image of a solid-state drive (SSD) to analyze its contents. To guarantee that the storage media remains unaltered and the evidence is admissible in court, what technical control should the examiner use when connecting the SSD to the forensic workstation?

- A) A software-defined network (SDN) tap
- B) A hardware or software write blocker
- C) A Syslog-ng server over TCP
- D) An in-memory memory dumper

## Question 3
During a criminal hacking trial, the prosecution attempts to introduce the server event logs to prove the defendant accessed confidential files. The defense lawyer objects, arguing that the logs are hearsay because they are computer-generated statements and cannot be cross-examined. Under which legal rule or exception can the prosecution argue for the logs to be admitted?

- A) Best evidence rule
- B) Business records exception to the hearsay rule
- C) Parole evidence rule
- D) Plain view doctrine

## Question 4
An IT department discovers a malicious script scheduled to run automatically on a primary domain controller. The script checks if a specific system administrator's user account is disabled; if the account is disabled, the script immediately deletes all company databases and backup drives. What type of attack motivation and threat profile does this script represent?

- A) Thrill attack; script kiddie executing a zero-day exploit
- B) Business attack; competitor conducting corporate espionage
- C) Grudge attack; disgruntled insider threat deploying a logic bomb
- D) Military attack; advanced persistent threat (APT) establishing persistence

## Question 5
A CISSP-certified security consultant discovers that a client's web application is vulnerable to SQL injection, exposing thousands of customer records. The client's CEO instructs the consultant to keep the finding confidential and not to publish it, as it would disrupt a planned venture capital funding round. According to the (ISC)² Code of Ethics, which canon must dictate the consultant's highest priority?

- A) Canon 3: Provide diligent and competent service to principals.
- B) Canon 1: Protect society, the common good, necessary public trust, and infrastructure.
- C) Canon 2: Act honorably, honestly, justly, responsibly, and legally.
- D) Canon 4: Advance and protect the profession.
