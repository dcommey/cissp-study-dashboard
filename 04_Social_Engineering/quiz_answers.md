# CISSP Practice Quiz Answers - Video 4: Social Engineering

Here are the correct answers and detailed explanations for the Video 4 quiz.

---

### Question 1
**Correct Answer: B**

*   **Explanation**: Piggybacking occurs when an authorized person actively helps or allows an unauthorized person to enter a secure area, typically preying on human politeness or helpfulness (e.g., carrying boxes). Tailgating occurs when an attacker slips in right behind an authorized user *without* their active knowledge or assistance.
*   **Why others are incorrect**:
    *   *A is incorrect* because tailgating does not involve the employee holding the door, and authority was not the primary driver (helpfulness was).
    *   *C is incorrect* because it was piggybacking, and consensus (social proof) was not the main driver.
    *   *D is incorrect* because scarcity (time limits) was not exploited.

---

### Question 2
**Correct Answer: B**

*   **Explanation**: Typosquatting is registering common typos or misspellings of popular domains (e.g., replacement of characters, missing letters) to catch users who mistype a web address.
*   **Why others are incorrect**:
    *   *A is incorrect* because pharming is technical redirection (DNS manipulation, DHCP, hosts file) where the user types the domain perfectly but still gets redirected.
    *   *C is incorrect* because a watering hole attack targets a third-party website frequently visited by the target group.
    *   *D is incorrect* because DNS spoofing is a network-level manipulation of DNS responses.

---

### Question 3
**Correct Answer: C**

*   **Explanation**: Finding sensitive printed information in trash bins is dumpster diving. The correct administrative control to prevent this is implementing a strict shred-all policy and deploying locked paper shredding bins.
*   **Why others are incorrect**:
    *   *A is incorrect* because shoulder surfing is watching screens directly, not picking up garbage.
    *   *B is incorrect* because hoaxes are false alerts sent online.
    *   *D is incorrect* because watering hole attacks occur on public websites.

---

### Question 4
**Correct Answer: C**

*   **Explanation**: Pharming occurs when a user enters the correct web address but is redirected to a malicious site. Modifying the local client 'hosts' file is a standard way attackers execute pharming campaigns.
*   **Why others are incorrect**:
    *   *A is incorrect* because watering hole attacks involve hacking a legitimate site that users visit, not redirecting their typed links.
    *   *B and D are incorrect* because phishing/spear phishing require the user to click on a link in a message, whereas pharming works even when the user types the correct address manually.

---

### Question 5
**Correct Answer: C**

*   **Explanation**: The attacker impersonated the CISO (exploiting the CISO's **Authority**) and demanded immediate action under threat of termination within 2 hours (exploiting **Scarcity / False Urgency**).
*   **Why others are incorrect**:
    *   *A is incorrect* because consensus (social proof) was not mentioned (no claim that 'everyone else is doing it').
    *   *B is incorrect* because the email was authoritarian and urgent, not using likability.
    *   *D is incorrect* because familiarity is about looking like a friend or contact, whereas CISO represents hierarchical authority.
